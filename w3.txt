5. 패턴 매칭
  - 프롬프트에서 셀, 프로그램이 작동한다.
  grep Linux sample.txt
  - 어느부분이 셀이 담담하고 어느 부분이 프롬프트가 담당하는지
  - 셸은 사용자와 리눅스 운영체제를 연결해주는 매개체
  리눅스 - 셸, 프롬프트 - 사용자
  - 셸의 역할은 명령 실행, 와일드카드 처리한다.

  grep Linux *.txt
  셸이 *.txt를 sample.txt, a.txt, 로 변경해주는 것
  - 셸이 패턴매칭을 처리하여 결과물을 프로그램에 보낸다.
  - head xxx | grep | ...
  - 셸이 이 파이프라인의 결과물을들 값으로 꺼내는 역할을 한다.

  - *
    - zero or more
    - *.txt : .txt, animals.txt, 등 txt파일 전부
    - 0개 이상의 연속열과 일치하는

  - ?
    - 1개 매칭
    ch1 ch2 ch3 ... ch100
    ex) grep Linux chap

  - []
    - 대괄호 안에 열거되어 있는 것 중 일치하는 문자열 1개 있을 때
    grep Linux ch[12345]
    grep Linux ch[1-5]

    Q. 짝수로 끝나는 모든 파일 (chxxx)
    grep Linux ch*[02468]
    Q. 대문자로 시작하고, 언더스코어 포함되어 있는 파이썬 파일 매칭
    grep Linux [A-Z]*_*.py
    
    Q. /etc 디렉터리 안에 파일 확장자가 .conf인 파일 목록 조회하기
    ls /etc/*.conf

    - 매칭되는 파일이 없으면 에러
      ls *.doc
    - 디렉토리 경로 파일명
